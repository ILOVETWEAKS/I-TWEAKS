# I TWEAKS

**I TWEAKS** is a professional Windows performance optimizer and system utility panel. Designed for gamers, power users, and anyone who wants complete control over their system, I TWEAKS offers a sleek neon-themed UI and easy-to-use performance tweaks.

---

## Features

### Dashboard
- Real-time CPU, RAM, Disk, and GPU usage with neon progress bars.
- FPS overlay toggle to monitor performance while gaming.

### System Utilities
- Clear temporary files safely.
- Restart Windows Explorer instantly.
- Activate High Performance or Ultimate Performance power plans.
- Toggle Game Mode for smoother gameplay.

### Background Apps
- View all running applications and processes.
- Manually terminate unnecessary apps safely.
- Refresh process list instantly.

### Tabs & Tools (Skeleton for future expansion)
- Registry Settings
- Network Settings
- Privacy Settings
- System Info
- Tweaks
- Apps
- Maintenance
- Setup
- About (logs and actions)

### UI
- Neon-themed professional design.
- Scrollable frames and grouped sections.
- Hover effects on buttons and interactive controls.
- Responsive multi-tab layout.
